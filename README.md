Thin Python [Blot're][blotre] client.

**Supports**
* Unauthenticated queries.
* Authorization flows.
 * Exchange authorization code for creds.
 * Disposable client creation and redemption.
* Authorized requests.
 * Automatic exchanges refresh token if available.
